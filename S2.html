Links

The <a> tag create a link. For example, 


<a href = "https://www.example.com" target = "_blank"> Click me! </a>

will create a link like this:

Click me!

This link will open in a new tab.
If would like it to open in the same tab, you could change the "target" attribute to "_self": 


<a href = "https://www.example.com" target = "_self"> Click me! </a>

The output would then be:

Click me!

There are two* other values for the target attribute: _parent, _top. These are used when you have links inside iframes. _parent will open the link in the context that the frame was embedded in. _top will open the link in the topmost browsing context in the case of nested iframes.

Image source

As you can see, clicking the link will bring you to the website defined in the "href" attribute. You can change this to any site you want. In fact, it doesn't even have to point to a webpage. It can also link to other url schemes:

  • "javascript:" will run javascript code. This is generally bad practice. Example

  • "tel:" will call a number. Example

  • "mailto:" will send an email. Example

  • "sms:" will send a text message. Example

Links may also be used to skip to other parts in the same page. You can see this in the table of contents at the top of this page. For example, this link will bring you to the top of this section.

There are many other link attributes not covered here. You can read more about them online.
