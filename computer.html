Links

The <a> tag create a link. For example,


<a href = "https://www.example.com" target = "_blank"> Click me! </a>


will create a link like this:

Click me!

This link will open in a new tab.
If would like it to open in the same tab, you could change the "target" attribute to "_self":


<a href = "https://www.example.com" target = "_self"> Click me! </a>


The output would then be:

Click me!

There are two* other values for the target attribute: _parent, _top. These are used when you have links inside iframes. _parent will open the link in the context that the frame was embedded in. _top will open the link in the topmost browsing context in the case of nested iframes.

Image source

Here's a table with the different values for the target attribute:

Target Opens in...
_self Same tab
_blank New tab
_parent Upper context
_top Topmost context

As you can see, clicking the link will bring you to the website defined in the "href" attribute. You can change this to any site you want. In fact, it doesn't even have to point to a webpage. It can also link to other url schemes:

  • "javascript:" will run javascript code. This is generally bad practice. Example

  • "tel:" will call a number. Example

  • "mailto:" will send an email. Example

  • "sms:" will send a text message. Example

Links may also be used to skip to other parts in the same page. You can see this in the table of contents at the top of this page. For example, this link will bring you to the top of this section.

Images

Images can be displayed using the <img> tag. You can specify the path to the image in the "src" attribute. For example:



<img src = "/assets/htmlLogo.png" alt = "Example image" width = "400vw">

will render as:

Image source

You are very highly recommended to use the "alt" tag. This tag is used by screen readers for accesibility. It will also appear if your image fails to load. You can specify the width and height of the image using the "width" and "height" attributes.
You can use images as links by nesting them inside an anchor tag:

<a href = "https://www.example.com" target = "_blank">
    <img src = "/assets/ocean.jpg" alt = "Example image" width = "400vw">
</a>


Image source

You can use any of the following image types in html:

  • APNG
  • AVIF
  • GIF
  • JPEG
  • PNG
  • SVG
  • WebP

Audio

Note: Example music used throughout this section is from https://www.no-copyright-music.com/cinematic/. The pop sound effect is from https://mixkit.co/free-sound-effects/pop/

You can use the <audio> tag to add sound to your webpage. You link to the sound file using the "source" tag like this:

<audio controls>
   <source src = "./assets/music_example.mp3">
</audio>

You can use the "loop" attribute to make your sound automatically repeat. You can also use the muted attribute to have it start with no volume.

An example with loop enabled

An example with muted enabled

Music that autoplays when the page is loaded is usually blocked by the browser. However, audio can be played upon user interaction through Javascript.

Video

You may embed video into your HTML site using the <video> tag.

<video controls width="500vw">
   <source src = "assets/oceanVideo.mp4">
</video>

You can see that, similar to the audio tag, the video tag also uses the "controls", "width", "loop", and "height" attributes and the "source" tag. For videos, autoplay is allowed as long as you specify the "muted" attribute. Autoplay with sound is not allowed in most browsers.

<video controls muted autoplay loop width="500vw">
   <source src = "assets/oceanVideo.mp4">
</video>
